Quên mật khẩu

Nhập tài khoản admin (VD: admin). Mã 6 số gửi tới email admin trong hệ thống.

Import sản phẩm

Chọn file .csv hoặc .xlsx. Dòng đầu là tiêu đề cột. Màu/size nhiều giá trị dùng dấu ; (VD: Trắng;Đen). Tồn kho: S:10;M:8. Danh mục: ao, quan, vay, ao-khoac.
